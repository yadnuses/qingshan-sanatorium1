# 《青山疗养院》通关攻略

## 游戏简介

你叫**周明远**，是一名调查记者。三个月前，你的妹妹在**青山疗养院**离奇失踪。警方调查无果，院方坚称"从未有过此人"。但你找到了妹妹的日记，里面记录着令人不安的内容...

**游戏类型**：文字解谜、剧情探索
**核心机制**：在页面中寻找可疑关键词，输入搜索框解锁隐藏内容
**结局数量**：4个（3个普通结局 + 1个真结局）

## 游戏技巧

1. **注意红色文字**：页面中红色标记的都是可搜索关键词
2. **多尝试别名**：一个关键词可能有多种叫法
3. **查看页面源码**：有时线索藏在HTML注释中
4. **记录已解锁页面**：游戏有50个秘密页面
5. **不要跳过剧情**：每个页面都包含重要信息
